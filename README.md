# Segmentation Model CherryNet

Semantic image segmentation project using deep learning models implemented in PyTorch.

## Current Progress
- Custom PyTorch Dataset & DataLoader
- U-Net implementation from scratch
- GPU training support
- Binary segmentation training pipeline

## Tech Stack
- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib

## Features
- Custom semantic segmentation pipeline
- Training loop with BCEWithLogitsLoss
- GPU acceleration using CUDA
- Modular architecture for adding multiple segmentation models

## Future Plans
- Add multiple segmentation architectures
- Compare model performances
- Add Dice Score and IoU metrics
- Visualization of predictions
- Data augmentation
- Validation pipeline
